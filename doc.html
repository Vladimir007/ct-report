NAME

./commit-tester-report.pl: The program generates commit-tester html report form txt results.


SYNOPSIS
./commit-tester-report.pl [option...]
OPTIONS
-o, --result-file
is a file where results will be put in html format. If it isn't
specified then the output is placed to the file 'commit-tester-results.html'
in the current directory. If file was already existed you will be
asked if you want to rewrite it.
-h, --help
Print this help and exit with an error.
--files=" ..."
and next files are commit-tester txt results.
It can be found at commit-tester-results/
--rewrite
Do not ask confirm of html file rewriting if it already exists.
In results table yellow color means Time limit; green - Memory limit.
